# Campus Network Design Checklist

## Overview

This checklist provides a structured approach for designing scalable, resilient, and secure enterprise campus networks.

## Planning

- Define business and technical requirements
- Estimate user and device growth
- Identify critical applications
- Determine availability requirements

## Physical Design

- Core, Distribution, and Access layers
- Redundant uplinks
- Fiber and copper media selection
- Rack space and power planning

## Layer 2 Design

- VLAN design
- Trunk configuration
- Rapid STP or MST
- EtherChannel/LACP

## Layer 3 Design

- IP addressing plan
- OSPF or EIGRP design
- Route summarization
- First Hop Redundancy (HSRP/VRRP)

## Security

- 802.1X authentication
- ACL implementation
- DHCP Snooping
- Dynamic ARP Inspection
- Port Security

## Wireless

- Access Point placement
- RF coverage validation
- SSID planning
- Guest network segmentation

## Monitoring

- SNMP
- Syslog
- NetFlow
- NTP
- Configuration backups

## Documentation

- Network topology diagrams
- IP address inventory
- Device inventory
- VLAN assignments
- Change records

## Best Practices

- Design for redundancy
- Standardize configurations
- Validate before production deployment
- Keep documentation up to date
