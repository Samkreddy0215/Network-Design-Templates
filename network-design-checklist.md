# Network Design Checklist

## Planning
- Gather business and technical requirements
- Identify scalability requirements
- Define security requirements
- Document application dependencies

## Network Architecture
- Create high-level design (HLD)
- Create low-level design (LLD)
- Define IP addressing scheme
- Plan routing protocols (BGP, OSPF, EIGRP)

## High Availability
- Redundant core switches
- Dual WAN connectivity
- Redundant firewalls
- Load balancing requirements

## Security
- Network segmentation
- Firewall policies
- NAC/ISE requirements
- VPN requirements

## Monitoring
- SNMP monitoring
- Syslog configuration
- NetFlow/sFlow
- Alerting and reporting

## Documentation
- Network diagrams
- Device inventory
- Configuration standards
- Disaster recovery procedures
